# Bulk RNA-seq Analysis of CD4⁺ T Cells

This repository contains the analysis of **bulk RNA sequencing data** from human CD4⁺ T cells, originally generated by **Cano-Gomez et al., Nature Communications 11, 1801 (2020)**. The dataset explores transcriptional responses of CD4⁺ T cells to cytokine stimulation.

---

## Dataset Description

This analysis uses **bulk RNA sequencing data** from human CD4+ T cells.

**Raw Counts (`RNA_seq_raw_counts.txt.gz`)**  
- **Size:** 58,051 genes × 94 samples  
- **Format:** Rows are genes (Ensembl IDs), columns are samples. Values are raw read counts.  

**Metadata (`bulk_RNAseq_metadata.txt.gz`)**  
- **Size:** 94 samples × 10 variables  
- **Variables:**  
  - `sample_id`: Sample identifier  
  - `cell_type`: CD4⁺ T cell subset (Naive, Memory)  
  - `cytokine_condition`: Cytokine stimulation condition (IFNB, Th0, Th2, Th17, Resting)  
  - `stimulation_time`: Duration of stimulation (16h, 5d)  
  - `donor_id`: Donor identifier  
  - `sex`: Donor sex  
  - `age`: Donor age  
  - `sequencing_batch`: Sequencing batch number  
  - `cell_culture_batch`: Cell culture batch number  

> The dataset allows analysis of gene expression differences across cell types, stimulation conditions, and donors, while controlling for batch effects.

---

## Analysis Workflow

1. **Loading and Pre-processing**
   - Load raw counts and metadata into R
   - Ensure sample IDs in metadata match expression matrix columns
   - Filter out lowly expressed genes  

2. **Normalization and Transformation**
   - Size-factor normalization with DESeq2  
   - Variance-stabilizing transformations (rlog, vst) for visualization  

3. **Exploratory Data Analysis**
   - Principal Component Analysis (PCA) to explore sample clustering  
   - Detect batch effects or outliers  

4. **Differential Expression Analysis**
   - Use DESeq2 for identifying differentially expressed genes (DEGs)  
   - Volcano plots highlight significant fold changes and p-values  
   - Heatmaps show DEG expression patterns across samples  

5. **Visualization**
   - Volcano plots, heatmaps, and PCA plots for interpreting transcriptional changes
